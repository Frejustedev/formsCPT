FAQ

Questions fréquentes

Mes données sont-elles vraiment sécurisées ?

Oui. L'application ne fait aucune requête réseau. Toutes les données vivent dans un fichier `registre.json` à côté de l'.exe (Windows) ou dans le sandbox sécurisé de l'app (Android/iOS). Vous êtes propriétaire du fichier, vous le copiez où vous voulez.

Comment partager les données entre plusieurs postes ?

Utilisez l'export / import JSON depuis le panneau Administration. Vous pouvez aussi placer le `registre.json` sur un dossier réseau partagé (l'app ne supporte pas l'écriture concurrente, donc un seul utilisateur à la fois).

Pourquoi pas de cloud ou de compte ?

Choix délibéré. Les données médicales sont sensibles ; les héberger dans le cloud ajoute des contraintes réglementaires (HDS, RGPD, consentements) sans valeur ajoutée pour un registre individuel ou de service.

Puis-je modifier les champs ou les choix prédéfinis ?

Le code est open source. Toutes les listes sont centralisées dans un seul fichier (`lib/options.ts`) et le schéma dans `lib/schemas.ts`. Modifier, recompiler, redistribuer.

Y a-t-il un audit-trail ?

Oui. Chaque modification d'un dossier crée un instantané daté de l'état précédent (`record_versions`). Les exports / imports / créations / suppressions sont aussi journalisés (visible dans Administration).
